Ian Darwin's Java Templates

This is my current list of Java template files. Any of these can be used for any purpose, without credit - they're too small and too obvious to bother copyrighting.

All of these have CVS tags already in place (you need to change one character, which is to prevent the tags from being expanded while I work on the templates themselves).

  • @.txt, just the @author/@version identity template
  • AppApp.htm, an Applet/Application - demo HTML
  • AppApp.java, an Applet/Application - Java source
  • Applet.java, a simple applet
  • G2D.java, a Component with Graphics2D use
  • GUI0.java, a GUI constructing a Frame
  • GUI1.java, a GUI subclassing Frame
  • main.java, a line-mode main program
  • Swing.java, a GUI based on SWING
